# Tag Properties

 The properties  that a tag contains depend on the type of tag that the user creates and the configuration that the user enables.

 On the Add page, only the Basic Information and Valus configurations are displayed by default. All other configurations are not enabled. Only when the corresponding configuration is enabled, the corresponding configuration information will be displayed on the Add page  in the following figure.

 Note:If you configure a configuration item and then cancel it and save the tag. The next time the configuration is enabled again on that tag, the result of the last configuration will be displayed automatically.

**Note: When the tag Data Type is selected as Double，the system only supports a maximum of 15 decimal places.**

## **I/O Tag**

 I/O tags contain the following properties:

-  Basic I nformation
-  Value
-  Alarm
-  History
-  Deadband
-  Scale
-  Event
-  Simulate
-  Custom
