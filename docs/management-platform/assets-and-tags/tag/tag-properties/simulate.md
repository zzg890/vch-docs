# Simulate

The simulation of tags play an important role in VC Hub systems, and they are often used to simulate and represent continuity data related to actual processes.

The following are some common applications of analog properties in VC Hub systems:

- **Process Simulation**: Analog properties can be used to simulate actual process parameters such as temperature, pressure, flow, etc. This helps operators understand the behavior of the system, conduct training, and test and optimize control algorithms without the need and risk of actual operation.
- **Simulation**: Using the simulate, VC Hub system can create virtual environments for training operators and testing the performance of control systems. Such simulation environments can model a variety of scenarios, including normal operation, emergency shutdowns, and equipment failures, to improve operator response.
- **Data Analysis and Trend Forecasting**: The data generated by the simulation property can be used for data analysis and trend forecasting. By observing changes in the simulated data, potential trends and patterns can be identified to make more accurate predictions and decisions.

When a tag has simulation turned on, the data will be pushed using the simulated values, regardless of whether the data source is bound or not.

## **How to enable**

In the setting pop-up window of the tag, there is a switch for simulation at the top, turn it on to make simulation settings.

![alt text](25.png)

After opening, the pop-up window will automatically display the simulation configuration items, users can set the simulation data according to the actual needs, click on the **"** **OK** **"** button after finishing the settings.
