# DingTalk

Used to configure sending alarm notifications via DingTalk to specific groups or user accounts.

## Create WeCom Service

1. Click "**Alarming**" -> "**Alarm Notifications**" -> "**Services**" to open the “**Services**” list page.

![alt text](2.png)

2. Click the "Add" button. In the new pop-up window, select "DingTalk".

![alt text](37.png)

3. Click "Next" to enter the detailed configuration window. 

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/--GPPGo_O48l6zmkkrW8X3hryWqRzYEdKWM7qwqjZIg.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

In the “Send To” field, **DingTalk Group** is selected by default. To send notifications to individual users, check the **DingTalk Memeber** option. You may select both simultaneously.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/nA9BCUJKcD_CkcuPfEXroQmLTTi6HGsWX38NDO7ztOk.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

4. Once the settings are complete, click the 'OK' button to add this configuration data.

#### How to add a group robot

1. Click the settings button of an  DingTalk group 

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/cMzSMHc7Nx-RaQD4K-VbVoltkWl6ZnTbOFTe4v8WxOM.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

2. In the Group Settings window, click on “Bot”.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/vbOV4xAU822oteg_BcsV5hu_aIog-n4RPiA80RhWaqU.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

3. Click “Add Bot” (Note: Only group owners and group administrators can add bots).

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/2FQOrdfpAgqZRahq9q6BhKjkGzi02Omu4JxSpnoHE-Q.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

4. Click “Custom” 

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/hkrnO8Tu80OduX7F-rGOf6qehFFfYk73Ed3nXOoZZxg.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

5. Click the "Add" button

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/vfYiu89ROSUBz_--FOynH_mXR9uv9uibeJ8lV17mXR8.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

6. Add Robot

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/smZb8fNZlzVSf-EPq20TxY_gKYU_qv21sq2GE6ItMdQ.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

If “Additional Signature” is enabled under **Security Settings**, then when creating a DingTalk service and adding a DingTalk group, you must enter the robot’s generated signature key into the corresponding **Secret** field. If “Additional Signature” is not enabled, no secret needs to be provided.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/kGvLtoR0-QrgJRsWZYasgn4XbptjbIkSOTq7b13rY5c.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

| **Note**: Adding group robots is only supported on the desktop client; it is not supported on mobile. |

## How to view group robot Webhook URL and Additional Signature

In a DingTalk group, click the **Group Settings** button in the top-right corner. In the settings dialog, click on a specific group robot to view that robot’s Webhook URL and Additional Signature information.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/ET-4mCG1jCrrU4qPXYQPvyIHBXfHMIgJCVZkf9b9Np4.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

| **Note: Only group robots that created by yourself can view their Webhook URL.** |

#### How to create an app

1. Log in to the  DingTalk management  system (  [https://open-dev.dingtalk.com](https://open-dev.dingtalk.com/) )，Click **Create an app** on the “App Management” page.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/WDfgNdXE_59cmpKbIB8fZ5FQm-BqedjLqCzUbRQGwmM.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

2. Fill in and save your application information.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/oPWGDgkVgNsuq_k5G-zpaI01QLm8eVL8lXiSc8uHMRw.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

3. After saving, you will be redirected to the following page, click “View Version Details” at the top of the page to enter the details page.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/5ShfoIXbi_5JEjwVATGh3IqwuyC5nSxLL_rMBL2x4bQ.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

4. Click the edit button

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/aDpSGNZmkuxhdrJjzLml_WRaXuTMqOrjs5MbC4ALrdE.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

5. When you are finished editing, click ”Save” button

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/nyAIB0aFoT1iuUhhXvAyFLpe_uVJNSNA4g92Fs5pmSw.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

6. Click "Direct Publishing" to publish the application

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/6Ek5HdmA3oyJdtFw54QDQYBSs1g08BBoozqzAbsP750.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

7. You can view the created applications on the** Application Development** page

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/pU9jjS4DuiMZarkWNX7EXFp9SwITF29GlHqyo_swucM.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

8. On the “Application Development” page, click the “Application Details” button of the application to enter the Add Application Capability page.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/EopnSOkj3s7FQVJ_khLMCU7YxuXz9qgf2sjLK7a_0E8.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

9. Click the robot开启机器⼈配置，配置完成点击页面底部的”发布“按钮进行发布

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/O7Cp-iRRF4CdvtoCaI9rP1ApbDblhhBPZFGbOMnkL2g.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

10. Open **Robot Configuration** and click the “Publish” button at the bottom of the page to publish the configuration.

![img](https://docs.wagoscada.cn/wiki/api/wiki/editor/QHXVK91b/MWcvGvY4/resources/nxdigTJDFQ985cqkZTzEuidjrXho6Z9MnE6hBi88iFQ.png?token=W.1Co7O0-8nsJqNC2GCED541xs2pJB4xXRHZBdjNYMPxDWHcwyEyS0tdtZ5xLFq1Y7pSnfPN_r6z7OMmjQu9jHtK5mrA)

11. Configuration completed

## How to use the Email Notification Service

In the alarm notification rules, you will select the notification service.

1. Click on **"Alarming" -> "Alarm Notifications" -> "Rules"** to enter the notification rules list page.
2. Click the **"New"** button in the upper right corner of the list.
3. In the pop-up window, click the **'+DingTalk'** button to add a new Email notification rule.In the notification service dropdown, select the previously created notification service.
