# 系统要求

#### **推荐硬件配置**

 CPU:   x86 64位架构，4核心及以上

 内存：16GB

 硬盘：SSD硬盘或以上性能硬盘， 10G以上可用硬盘空间

**对服务器硬件配置的要求和客户项目需要接入的设备数量，通讯协议，采集点数，采集频率，历史存储要求，报警数量，脚本数量和复杂度等有直接的关系。**



#### **最低硬件配置要求**

 CPU:   x86 64位架构，4核心

 内存：4GB

 硬盘：SSD硬盘或以上性能硬盘，10G以上可用硬盘空间



#### 支持的操作系统

Windows 10/11

Windows Server 2019/2022 

Linux (支持主流的Linux操作系统, 实测和推荐版本Ubuntu 20.04) 



#### 不支持的操作系统

RHEL 7 及之前版

CentOS的所有版本



#### 支持的浏览器

Google Chrome

Microsoft Edge



#### 支持的数据库

Microsoft® SQL Server 

MySQL

PostgreSQL

SQLite

InfluxDB
