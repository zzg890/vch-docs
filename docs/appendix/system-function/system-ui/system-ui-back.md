# System.UI.back


## Description
Call this function to return to the previous page after System.UI.open().

## Grammar
System.UI.back(): void

Parameter

Nothing

Return

Nothing

## Code Example                                                                                                                                                                                                                                                                                                          
Execute the script System.UI.open('B') on Page A to open Page B.
```typescript 

System.UI.open('B');
```   
Execute the script System.UI.back() on Page B to return to Page A.
```typescript 

System.UI.back();

```
