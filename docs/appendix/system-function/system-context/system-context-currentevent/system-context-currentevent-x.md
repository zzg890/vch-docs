

# System.Context.currentEvent.x


## Description
The horizontal coordinate of the event relative to the page.

## Grammar
System.Context.currentEvent.x

## Code Example                                                                                                                                                                                                                                                                                                          
Obtain the horizontal coordinates of the current triggering event. 
```typescript 
const x = System.Context.currentEvent.x;
console.log(x)

```
