# System.Context.currentEvent.y


## Description
The vertical coordinate of the event relative to the page.

## Grammar
System.Context.currentEvent.y

## Code Example                                                                                                                                                                                                                                                                                                          
Obtain the vertical coordinates of the current triggering event. 
```typescript 
const y = System.Context.currentEvent.y;
console.log(y)

```
