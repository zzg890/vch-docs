

# System.Context.currentAlarm.nodeName


## Description

The name of the node where the current alarm is located.
## Grammar
System.Context.currentAlarm.nodeName

## Code Example                                                                                                                                                                                                                                                                                                          
Get the name of the node where the current alarm is located.
```typescript 

const node = System.Context.currentAlarm.nodeName;
console.log(node);

```
